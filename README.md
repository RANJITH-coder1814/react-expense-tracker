# react-expense-tracker
A simple Expense Tracker built using React JS. Users can add income and expense transactions, view total balance, and track transaction history. Data is saved in localStorage.

A simple Expense Tracker built using React JS.

## Features
- Add income and expenses
- Delete transactions
- Real-time balance calculation
- Data saved in localStorage

## Tech Stack
- React JS
- JavaScript
- CSS

## Setup

npm install
npm start
